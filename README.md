# Portfolio Website

Welcome to my personal portfolio website! This site showcases my skills, projects, and experience as a Full Stack Web Developer. Explore my work, learn about my background, and feel free to connect with me.

## 🚀 Features

- Modern, responsive design
- Project showcase with tech stack icons
- Downloadable resume
- Contact form
- Social media links

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase

## 📂 Project Structure

- `index.html` — Main HTML file
- `style.css` — Custom styles
- `index.js` — Main JavaScript logic
- `firebase.js` — Firebase configuration
- `src/` — Images and assets
